 🖥️ Calculadora JS

Uma calculadora simples feita com **HTML, CSS e JavaScript**!  
Ótima para praticar lógica de programação e criar projetos divertidos.

---

## 🚀 Tecnologias usadas
✨ HTML  
✨ CSS  
✨ JavaScript  

---

## 📌 Funcionalidades
- ➕ Soma  
- ➖ Subtração  
- ✖️ Multiplicação  
- ➗ Divisão  
- 🧹 Limpar tela  
- Interface simples e intuitiva  
- Responsiva para celular e computador
